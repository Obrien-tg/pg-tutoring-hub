For Administrators:
  1. Create a Firebase project at Firebase Console
  2. Enable Authentication, Firestore, Cloud Messaging, and Storage
  3. Generate a service account key and add it to your environment variables
  4. For bundlers/modern setups: Run npm install firebase and use modular imports instead of script tags
  5. Configure the following environment variables in your .env file:
FIREBASE_API_KEY=your_api_key_here
FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
FIREBASE_PROJECT_ID=your_project_id
FIREBASE_STORAGE_BUCKET=your_project.appspot.com
FIREBASE_MESSAGING_SENDER_ID=your_sender_id
FIREBASE_APP_ID=your_app_id
FIREBASE_VAPID_KEY=your_vapid_key
FIREBASE_SERVICE_ACCOUNT_KEY=/path/to/service-account-key.json
ENABLE_FIREBASE_FEATURES=True
Note: Firebase integration is optional. The platform will work with Django Channels for real-time features if Firebase is not configured.
